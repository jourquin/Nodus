Nodus documentation


Nodus 8.x is not provided with complete reference and user guides. It is however important that the user has a good understanding of the methodological framework the software is based on. This is particularly true for the ``Virtual Networks'' and the different cost functions they need. Therefore, several documents are provided in the "doc" directory, and a Nodus project can be found in the "demo" directory. The latest provides several examples on how to use the software.

To start with, a few screenshots can be visualized.

Methodological documents :

  • Methodological note : The basic concepts.
  • Virtual Networks : The first paper on Virtual Networks, published in Transportation Research C, in 1996.
  • Equilibrium assignments : A paper published in 2006 in the European Journal of Transport Infrastructure Research.
  • Multi-flow assignments : A paper published in 2005 in Studies in Regional Sciences.
  • Lines and services : A conference paper presented in 2008 at the European Transport Conference.
  • Semi-Dynamic Assignments : Chapter 5 of the PhD thesis of J. Piotte (2012, in French).
Beside these documents, several papers presenting research projects in which Nodus was used were published or presented in conferences. Most papers can be downloaded on Research Gate.
